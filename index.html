Chef IA - Generador de Recetas

Inteligencia Artificial · Herramienta Culinaria

Descripción

Un aplicativo web Full-Stack impulsado por Inteligencia Artificial (Google Gemini 2.5-flash) que analiza ingredientes sueltos ingresados por el usuario para crear recetas culinarias precisas y estructuradas de forma instantánea. Destaca por su seguridad al manejar la API Key desde un backend Proxy en Express, un diseño UI inmersivo construido únicamente con CSS Puro y Vanilla JS, y el aprovechamiento avanzado del Structured Output JSON de Gemini para garantizar resultados impecables en todo momento.

Funcionalidades principales

  • 🤖
    Generación IA por Gemini — Integración nativa con la API oficial de Google Gemini (2.5-flash).
  • 🧩
    Control de Estructura JSON — Utilización nativa del Structured Output (Response Schemas) forzando que la IA siempre devuelva la receta en un objeto JSON limpio y parseable.
  • 🛡️
    Backend Proxy en Node.js — Arquitectura segura con Express que protege y aísla la Gemini API Key, evitando su filtración en los navegadores de los clientes.
  • 🌗
    Sistema "Dark Mode" Integrado — Soporte automático de tema oscuro y claro sincronizado con la preferencia del sistema más un Toggle manual mediante localStorage.
  • 🎨
    Aesthetics y Componentes Modernos — Diseño UI inmersivo creado con CSS Puro (sin frameworks), usando tipografía Outfit y estructura de Cards.
  • ⌛
    Estados Asíncronos Robustos — Botones bloqueados contra clics múltiples, loaders spinners personalizados y desactivación de inputs originada dentro de promesas fetch().
  • 🚨
    Control de Errores Full-Stack — Flujos de mitigación ante API Keys inválidas o errores de comunicación con mensajes de alerta directos al DOM.
  • 📱
    Diseño 100% Responsive — Interfaz acomodable a pantallas de escritorio y móvil basándose en Flexbox y Grid CSS nativo.
  • ✨
    Implementación de Iconografía Ágil — Uso inyectable y optimizado mediante unpkg de la librería moderna Lucide Icons.
  • 🚀
    Despliegue Configurado para Render — Estructurado para facilitar un montaje continuo remoto sin modificaciones mediante la gestión correcta de rutas internas e inyecciones de puertos de entorno.

ControlGastos

Finanzas personales · Mercado peruano 🇵🇪

Descripción

ControlGastos es una aplicación Flutter de gestión de finanzas personales diseñada para el mercado peruano. Permite registrar gastos organizados por categorías con presupuesto mensual, monitorear el saldo disponible en tiempo real y recibir alertas inteligentes antes de superar los límites — todo sin conexión a Internet, con almacenamiento 100% local usando Hive (NoSQL embebido). Incluye un sistema de quincena (pagos en días 13 y 28 de cada mes), autenticación biométrica, exportación de reportes PDF y backup completo en JSON.

Funcionalidades principales

  • 📊
    Dashboard — Saldo real, tarjeta de quincena, cards de presupuesto por categoría con barra de progreso dinámica
  • 💸
    Registro de gastos — Con categoría, subcategoría, fecha y foto de comprobante (galería o cámara)
  • 🔍
    Búsqueda avanzada — Filtro por texto, categoría y rango de fechas dentro del mes
  • 📁
    Categorías personalizadas — Emoji, color, límite de gasto, subcategorías y categorías fijas
  • 📈
    Estadísticas — Gráfico de barras y dona mensual por categoría (fl_chart)
  • 🔔
    Alertas automáticas — Notificación local al 70% y 90% del presupuesto (umbrales configurables)
  • 📄
    Reporte PDF mensual — Con tabla de presupuestos y detalle de gastos, compartido desde el OS
  • 💾
    Backup completo — Exportar/importar JSON con todos los datos (categorías, gastos, ajustes)
  • 🔐
    Biometría — Login con huella dactilar o Face ID (con fallback a PIN)
  • 📆
    Lógica de quincena — Calcula automáticamente el próximo día de pago (13 o 28), ajustando fines de semana a viernes anterior
